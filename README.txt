Projeto de Ciência de Dados com o intuíto de fixar conceitos em Machine Learning.
O projeto é de referência ao livro "Projetos de ciência de dados com Python: Abordagem de estudo de caso para a criação de projetos de ciência 
de dados bem-sucedidos usando Python, pandas e scikit-learn". Autor: Stephen Klosterman

Aqui implemento alguns algoritmos de aprendizado de máquina e plotagens complementares aos propostos no livro.
